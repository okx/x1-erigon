# How to run
```shell
make build-docker;
cd test; 
# run full test
make run; 
# or run minimum test
make min-run;
# run all with bridge
make all;

```

# Important
To check the consistency state, turn on the following switch
``` shell
vim config/test.erigon.seq.config.yaml
# modify executor-strict to true
zkevm.executor-strict: true
```
